# Vpoint Calculator Platform

> Платформа для розрахунку вартості виробництва меблів

## Швидкий Старт

1. Запустіть `START.bat` для локальної розробки
2. Відкрийте `index.html` в браузері

## Структура Проекту

```
v2/
├── index.html           # Головна сторінка (Dashboard)
├── core/                # Ядро платформи
│   ├── calculator.html  # Калькулятор (runtime)
│   ├── admin.html       # Редактор матриці
│   ├── designer.html    # Редактор дашборду
│   ├── css/             # Стилі
│   └── js/              # JavaScript модулі
├── data/                # Конфігурації та схеми
│   ├── config_*.js      # Налаштування платформи
│   └── schema_*.js      # Схеми калькуляторів
├── assets/              # Статичні ресурси
│   ├── icons/           # Іконки
│   └── hints/           # Зображення-підказки
└── docs/                # Документація
```

## Технології

- **Frontend**: Vanilla JavaScript (ES6+), CSS3
- **Залежності**: Немає (zero dependencies)
- **Сумісність**: Сучасні браузери (Chrome, Firefox, Edge)

## Інтеграція

Для інтеграції в корпоративну систему:

1. Замініть `local_saver.py` на ваш API endpoint
2. Схеми можна зберігати в БД замість JS-файлів
3. Дивіться `docs/API.md` для деталей
